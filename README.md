# home-main-guide

## 项目简介

`home-main-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内容以静态网页形式组织，不涉及特定域名或网站，仅作为页面存储与展示的通用空间。

## 目录结构

```
home-main-guide/
├── pages/          # 存放独立的 HTML 页面文件
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本文件
```

- `pages/` 目录：每个子目录或文件对应一个独立的页面，页面之间无依赖关系。
- `assets/` 目录：按需存放页面所需的公共或私有资源文件。
- `index.html`：可选的根目录入口文件，用于展示页面列表或导航。

## 页面归档说明

- 所有 HTML 页面均为独立完整文件，可直接在浏览器中打开运行。
- 页面内容不针对任何特定域名或网站，仅作为技术演示或资料存档。
- 归档页面可能包含示例数据或占位内容，不代表真实信息。

## 维护说明

- 新增页面：在 `pages/` 下创建新目录或文件，确保页面引用资源路径正确。
- 更新页面：直接修改对应 HTML 文件或资源文件，保持结构清晰。
- 删除页面：移除对应文件或目录，同时检查是否有其他页面引用其资源。
- 建议为每个页面添加简要说明（可在页面内部注释或本 README 中记录）。

## 注意事项

- 仓库不提供自动部署或构建脚本，页面以原始 HTML 形式存储。
- 请勿在页面中写入敏感信息或外部链接依赖，以免影响本地使用。
- 本仓库仅作为静态文件归档，不承担页面内容审核或更新职责。
